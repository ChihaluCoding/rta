# RTA Mod 仕様書

## スコープ
- RTA/IGTタイマー、バイオーム名、座標換算、エンド内クリスタル数をHUD表示する。
- ModMenuからRTA補助機能を切り替えられるようにする。
- エンドラ討伐時にクリア時間を表示し、`RTA_History` に履歴を保存する。
- `/rtahistory` コマンドを提供する。
- ポーズ画面にシード値リセットボタンを追加する。

## 制約
- 対象は Fabric / Minecraft `26.1.2` / Java 25。
- リプレイ機能は実装しない。
- 有料APIや外部サーバーは使用しない。
- ユーザー向け文言は日本語にする。

## 受け入れ条件
- JDK 25で `./gradlew.bat build` が成功する。
- `config/rta.json` に設定が保存され、ModMenuから変更できる。
- ワールド初回参加でタイマーが開始し、死亡またはエンドラ討伐で停止する。
- エンドラ討伐時にタイトル、サウンド、履歴保存が行われる。
- `/rtahistory` で保存済み履歴を表示できる。
- 設定ON時に一括破壊、自動精錬、確定ドロップ、エンダーアイ破壊無効、エンダーマンブロック持ち禁止が動作する。

## 非対象
- リプレイ録画・再生。
- 外部サービス連携。
- 既存ワールドのシードそのものを書き換える機能。
